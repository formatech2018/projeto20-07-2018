	Readme
	Requisitos para uso do sistema:
	
	1- Servidor com xampp ou wampp
	2- PHP na versão mínima 7.1
	3- MySQL na versão mínima 5.6
	4- Navegador Google Chrome, Mozilla Firefox ou Microsoft Edge

	Para utilizar o projeto:

	Passo1
	Copiar a pasta do projeto library para htdocs 
	no caso de usar o xampp ou www se usar o wampp

	Passo 2
	Criar um virtualhost com endereço de domínio: www.libraryitego.com.br

	Passo 3
	Inicie os serviços do apache e do mysql

	Passo 4	
	Importe o banco de dados da pasta 
	libraryitego-master/docs/Diagrama_fisico/SGBD/modelo_fisico_completo
